## üöÄ Quick Start

### 1. Installation

```bash
# Clone the repository
git clone https://github.com/ToxicSpawn/Pinnacle-AI.git
cd Pinnacle-AI

# Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`

# Install dependencies
pip install -r requirements.txt

# Install optional dependencies for full functionality
pip install -r requirements-dev.txt
```

### 2. Configuration

Copy the example configuration file and add your API keys:

```bash
cp config/settings.yaml.example config/settings.yaml
# Edit config/settings.yaml with your API keys
```

### 3. Running Pinnacle AI

**Interactive Mode** (recommended for exploration):
```bash
python main.py --interactive
```

**Single Task Mode**:
```bash
python main.py "Your task here"
```

**Benchmark Mode** (to evaluate performance):
```bash
python main.py --benchmark
```
